# TicketMoa

### Description
> TicketMoa의 front-end를 담당하는 application입니다.
> 
> Nginx를 Load Balancer로 활용하여 가용성을 높였습니다.

### Usage
1. .env 파일 생성
```
REACT_APP_HOST=localhost                   
```
2. 실행 
```bash
npm install react-scripts --save
npm run build
npm serve -s
```
